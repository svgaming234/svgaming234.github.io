FAQ:

Q: What is a user agent?
A: A user agent is information about your browser and device that is given to every website you visit.

Q: How can I prevent websites from seeing my real user agent?
A: There are many browser extensions that can change your user agent to something that is not your real user agent.

Q: What do websites use user agents for?
A: For example: some websites you download software from have multiple software versions for different operating systems. They use user agents to detect your operating system and give you the version of the software for your OS.
