Hello everyone! Today I will be teaching you some basics of using Git.
What is Git?
Git is a collaboration tool for writing code with your friends or colleagues.
Programs on Git are in repositories (repos) that can be hosted on various sites such as GitHub.
Well, how do I use it?
To start off, you can try cloning a reopository.
To clone a repository, type: git clone *repository URL*
To make a repo, you need to go to your Git hosting site of choice (usually GitHub or GitLab) and make a repository there.
After that you can clone it as shown above.
Next you can make some changes such as creating/deleting or editing files and then you can upload the changes with these commands:
git add .
git commit -m "*summary of changes*"
git push origin *branch name*
A deafult branch name is most likely master or main.
Made any changes on another computer? Type git pull to pull the changes onto your computer.
